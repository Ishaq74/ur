Randonnée alpine vers le plus haut sommet autour du lac

Description du parcours

Ascension exigeante vers le sommet emblématique de La Tournette (2351m), plus haut sommet dominant le lac d'Annecy. Vue spectaculaire garantie pour les randonneurs expérimentés. Passage en terrain rocheux.
